HBChecker

A python script that checks something (ex. A DSi's SD card) to see if specific files are missing or corrupted

It is currently tested for:

  • HiyaCFW v1.3.2 (11/6/2018 update)
  • TWiLight Menu++ v6.8.1.2

To Run:

  1. Place HBChecker.py or HBChecker.exe and HBCheckerItems.json on the root of your SD card
    • Windows: Run HBChecker.exe (Note: This may be falsely detected this as a virus, it is not, it's just a false positive)
    • Linux/macOS/other: Run HBChecker.py using Python 2 or 3
  3. With the default HBCheckerItems.json it will check for HiyaCFW and TWiLight Menu++ (But not for corruption of TWLMenu, see steps below)
  4. Make sure HiyaCFW Main, HiyaCFW [your region], and TWLMenu++ are found
  5. If all are found and it tells you No files were missing! then you should be good to go!
  6. If any weren't found, enter the corresponding number to manually check for it

If you want to check TWiLight Menu++ files for corruption:

  1. Download AltHBCheckerItems.zip
  2. Remove HBCheckerItem.json from the same directory as HBChecker.exe / HBChecker.py
  3. Open HBChecker.py / HBChecker.exe and drag/drop HBCheckerItems (TWLMenu CRC).json onto it
  4. Make sure TWLMenu++ 6.8.1.2 was found
  5. If is was found and it tells you No files were missing! then you should be good to go!
  6. If it wasn't found, enter 1 to manually check for it
